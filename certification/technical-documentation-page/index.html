Introduction

dockVind's suite of powerful website generation tools is a must have in every business's tech stack. This documentation will walk you through the various offerings dockVind provides that will bring your business upto speed in today's rapidly changing tech ecosystem.

These are the products currently offered in the dockVind Generatrix Suite

  • Survey Form
  • Tribute Page
  • Technical Documentation
  • Product Landing Page

Use the following command to install and set up dockVind on your system

$ sudo apt vind

Survey Form

Know the pulse of your audience using the Generatrix Suite Survey Form. The Generatrix Suite makes it easy to generate an aesthetic looking survey form in seconds.

Set up a Survey Form project using the following command

$ vind make survey

Tribute Page

Create a legacy for icons that command attention. dockVind's Tribute page templates are easy to set up and adapt to the person they are made for.

Set up a Tribute Page project using the following command

$ vind make tribute

Technical Documentation

Automatically generate beautifully formatted and easy to read documentation for your projects with our Technical Documentation generator.

Set up a Technical Documentation project using the following command

$ vind make documentation

Product Landing Page (Early Access)

dockVind is currently working on a powerful Product Landing page generator that is on par with its suite of tools.

You can test the Product Landing page generator to try it out before release. Set up a Product Landing Page project using the following command

$ vind make product
